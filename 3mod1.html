Application & Reflection:

The lessons learned in this exercise are useful in both spatial and aspatial data analysis. First, proper sampling techniques always need to be considered with analysis in mind. Second, data can lie. In this example it is very obvious that the interpolation of the second sample was generating a false positive result. Meaning, the analysis was done correctly, but improper sample data was used creating a result that is false. However, occurrences like this will not always be so obvious. For example, it would be much more difficult to spot with less extreme topographical variation in the data.
