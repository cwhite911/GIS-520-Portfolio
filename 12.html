Image Classification

Problem:

Classify land characteristics at Black Water National Wildlife Refuge from aerial photography.

Return to Top


Analysis Procedures:

The strategies used to solve this problem include image classification techniques and adjustments to aerial photography for easier interpretation of the landscape. Image classification produces a raster where each cell value represents a pixel value from the source image. Using tools such as supervised classification in ArcMap make it easy to explore data and create better sample datasets. Aerial photography can be made easier to understand by adjusting the band settings. For example, in a four band, the image inferred can easily show the difference between vegetation and other surfaces.

To create an image classification raster, start by refining the aerial photography so that it is in its most clear format for the task. This can be done by setting the bands on a RGB image, or by using other tools in ArcMapâ€™s Image Analysis. Once the source imagery is processed, use the Image Classification tool bar to create sample polygons for each classification type. The amount of samples used is variable, but use an amount that creates an accurate reflection of the landscape. Now that the samples are set, run the supervised classification tool. This will produce a raster that has been classified using the source samples as training. To improve the output raster add more or better placed source polygons and compare the results with the previous attempt.

Application & Reflection:

Image classification is a powerful tool that has many applications. Its main power comes from its ability to take away the tedious task of trying to manually classify everything in an image. Although it is not perfect, the using image classification tool greatly seeds up the analysis of images. Another example of how image classification is used is training the program to look for just one specific type of object, such as a fire hydrant or a cemetery headstone. Using image classification in this way can greatly speed up the digitization process of an editor and help create a base layer to begin with. Image classification allows aerial imagery to not just provide import background information, but to also provide quantitative data about the landscape itself.
