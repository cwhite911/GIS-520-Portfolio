Assignment 11:
Suitability Analysis and Weighted Overlay

  • Problem
  • Analysis Procedures
  • Results

Problem:

Determine what areas in the Great Smoky Mountains National Park are suitable for black bears basic needs where there is a low chance of human interaction.

Return to Top


Analysis Procedures:

The strategy used to solve this problem involves the weighted overlay tool in ArcMap. To use a weighted overlay all data must be in raster datasets of the same cell value and classification schemas. Also, the importance of each variable must be predetermined in order to perform useful analysis because this will provide the weights each variable is given during the analysis.

In order to perform weighted overlay analysis, all data must be converted into raster datasets of the same cell size (Figure 1). This can be done by using tools such as Feature to Raster or Euclidean distance, and if data needs to be created from one raster dataset (elevation to slope) this should be done now. Once all data is in its proper raster format, the datasets should be reclassified so that all the data sets share the same classification schema (generally classified in divisions of 3, 5 or 9). The data is now prepared to be run with the weighted overlay tool. In the tool parameters specify the weights associated with each raster dataset and set the new values if needed.

Application & Reflection:

The weighted overlay tool is a powerful method for creating suitability maps within defined areas. Beyond just ecological modeling the same principles can be applied to finding a new store location or even where water mains my break. For example, to determine water main breaks certain environment factors may be used to determine which areas a break is likely to occur. Factors could be age of pipe, material, soil type and many more. These could then be weighted by which factors are known to be directly related to mains breaking and create a model to predict where water mains may break in the future. This is important because it can help a water company know where and how to allocate money for repairs so that major main breaks can be prevented.
