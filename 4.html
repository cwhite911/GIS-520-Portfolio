Geocoding Tabular Data

Assignment 4: Geocoding Tabular Data
Geocoding is a method used to take spatially discriptive qualities and assign them values on a coordinate system.

Problem:

Identify possible areas of focus for a new advertising campaign for a local commercial establishment in Raleigh, NC.
The goal of the analysis is to determine unrepresented areas where new customers can be targeted in Wake County.

Analysis Procedures:

The main strategy used in the analysis is to geocode customer location data. This data must contain geographic information,
such as a full street address, but the data does not need to be at rooftop precision and can be as vague as 5-Digit ZIP Codes.
However, more accurate data improves the results of the analysis. For this analysis two geocoding styles will be used: US Address

Application & Reflection:

Geocoding is a powerful tool in GIS, and the ability to create address locators is an essential function of its power.
The reason for this is that not all dataset line up and there is no one size fit all approach to addresses.
New addresses are created and removed every day, and different organizations manage the incoming information producing
results that do not always line up with one another. For example, a local government might maintain address points for a city
or county, and with that address data everything from routing emergency personnel, to tracking utility accounts and billing is done.
If that local government tried to use Google maps for this role they would very quickly run into problems and accuracy issues.
These problems arrive because of the complexity behind address data and the services it helps provide. An example of how address data
can very quickly become complicated is with 911 call centers. Both the phone company and the organization responsible for taking the
emergency calls must be working with at least joined datasets that are regularly checked for errors. This is because the address associated
with the phone number might not always line up with the address in the call center database. The differences need to be reconciled so that
the emergency responders will arrive at the correct location.
