Results:


The result of this analysis provides a view of suitable locations for potential vineyard sites. The criteria determining the location are as follows. First, aspect must be between 157.5 and 247.5. Second, the slope must be between 1.5 and 15 percent. Third, the elevations must be above 200, and finally fourth, the site must be within one mile of a highway. The resulting layer is where all of these criteria are true. Map 1 shows each individual requirement and the final view of best suitable vineyard locations.

Application & Reflection:

Similar analysis can be used to narrow down the best location for a new radio tower, determining a habitat, or any other analysis that requires the measurement of environmental factors. This type of analysis is so powerful because it can be applied by many different disciplines. From land developers, ecologists, archaeologist, government, municipalities, etc. This approach allows the concatenation of many variables using Boolean logic.
