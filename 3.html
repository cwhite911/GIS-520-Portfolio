Problems:

Module 1: Basics of Data and Information
Exercise:Explore how sampling scheme can affect spatial interpolation
The problem presented in the exercise “Explore how sampling scheme can affect spatial interpolation” is to compare two different set of sample points from the same area that were taken with different sampling techniques (Random & Systematic), and compare the differences after they are interpolated.
Module 2: Cartography, Map Production, and Geovisualization
Exercise: Simplify a line feature
The problem presented in the exercise “Simplify a line feature” is to take complex geometry of a coastline and simplify it.
Module 3: Query and Measurement
Exercise: Working with measurements
The problem this exercise presents is to locate proper site locations for potential vineyards based on geospatial measurements.
Module 4: Transformations and Descriptive Summaries
Exercise: Use and compare Inverse Distance Weighting (IDW) and Kriging
Compare elevation surfaces that were created using Inverse Distance Weighting (IDW) and Kriging methods from the same set of sample points.
Module 5: Optimization and Hypothesis Testing
Exercise: Find the least cost path for a power line
Find the least cost path for a new power line between a fictional power station and substation.
Module 6: Uncertainty
Exercise:Examine Root Mean Squared Error
Transform digitized centerlines to real-world coordinates using control points.
