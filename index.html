NCSU GIS 520 Fall 2013

Advanced Geospatial Analysis

This course explores the different analytical techniques used to explore geospatial data with ArcMap 10.1. The course begins by looking at geospatial data issues and goes through both vector and raster analysis and ends with basic customization of the ArcMap interface. This class does not look at coding or other open source options that can be used. However, the course does do a good job covering the basic areas of analysis done with GIS and is a great introduction into even more advanced techniques to perform geospatial analysis.
